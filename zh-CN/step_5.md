## 无限动画

让我们来使动画永远重复进行。



+ 如果你希望太阳升起后落下，只需向你的动画添加更多关键帧：

    ```
    @keyframes sunrise {
        0%  
        33% 
        66% 
        100%
    }
    ```

    这表明动画以太阳位于天空的底部开始和结束，并在动画的 33% 至 66% 之间停留在顶部。

+ 现在你只需向 `#sun`（太阳）动画添加 `infinite`（无限）这个词，来使动画无限循环：

    ![screenshot](images/sunrise-infinite.png)

+ 测试你的动画。太阳是否不断升起和落下？
