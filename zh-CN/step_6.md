## 对天空进行动画处理

动画不仅仅可以移动。让我们对天空进行动画处理，使其在夜晚变暗。



+ 向你的 CSS 添加名为 `sky`（天空）的动画：

    ```
    @keyframes sky {
        0%
        100%
    }
    ```

    请注意这次你是对天空的颜色进行动画处理，而非位置。

+ 向你的天空添加代码，告诉它使用你的新动画：

    ```
    animation: sky 10s;
    ```

    ![screenshot](images/sunrise-sky.png)

+ 点击 **Autorun**（自动运行）来测试你的动画。
